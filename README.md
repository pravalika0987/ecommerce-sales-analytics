## Problem Statement
Analyze sales performance, customer behavior, and delivery
patterns from Brazil's largest e-commerce platform — Olist,
using 8 relational datasets with 100,000+ records.

## Dataset
- 8 relational datasets with 100,000+ orders
- Data: customers, orders, products, sellers,
  payments, reviews, geolocation
- Source: Olist Brazilian E-Commerce (Kaggle)

## Tools Used
| Tool | Purpose |
|---|---|
| Python + Pandas | Data integration and cleaning |
| SQL | Relational data querying |
| Power BI | KPI dashboards |
| Matplotlib + Seaborn | EDA visualizations |
| Jupyter Notebook | Analysis environment |

## Approach
1. Loaded and integrated 8 relational datasets using Pandas
2. Data quality profiling across all 8 tables
3. Missing value analysis and cleaning pipelines
4. EDA across order lifecycle and payment patterns
5. ETL pipeline implementation
6. Interactive Power BI KPI dashboards for stakeholders

## Key Results
- Identified delivery performance patterns across regions
- Uncovered revenue drivers and top-performing sellers
- Built dashboards presenting KPIs — total revenue,
  order volume, seller performance, and regional
  customer distribution
